Admin – Mitarbeiter IDs (Hybrid)

Ziel: AUTO_* IDs auf echte Mitarbeiter-IDs umhängen (inkl. Staffplan/Zeiten/Pausen).

AUTO_* umhängen

Vorgehen: alte AUTO_… auswählen → neue echte ID eingeben → „Umhängen“. Wenn die neue ID schon existiert, wird zusammengeführt (optional „alte Daten übernehmen“).

Mitarbeiterübersicht

Hinweis: In der Liste siehst du auch Dubletten (gleicher Name, unterschiedliche IDs). Das ist normal – genau dafür ist das Umhängen da.
